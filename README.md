# Hi, I'm Xiaoming

My current project is [Xiaoming Agent Skills](https://github.com/smwswk/xiaoming-agent-skills), a public catalog of 33 installable skills for Codex, Claude Code, and other agents that use the Agent Skills format.

I care about a few unfashionable details: clear stop rules, evidence before claims, visible failures, and an output a person can inspect.
